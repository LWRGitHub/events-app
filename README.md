# Events App

This is an app where you can find, create or go to an even. There is a DB of guests & events, have fun!


## Technology Used

- Flask
- Jinja2
- HTML/CSS
- SQL
- dotenv


## Install/Setup

### Install
```
pip3 install -r requirements.txt
```

### Setup
Make an `.env` file & add:
```
SQLALCHEMY_DATABASE_URI=sqlite:///database.db
```

### Run:
```
python3 app.py
```
